# JhCordovaVueDemo
Cordova+Vue Hybrid App

- 代码分两部分：`Vue`项目和`Cordova`项目
- 需要先打包Vue项目`npm run build`（打包文件直接生成在Cordova项目的`www`文件中）
- 在Cordova项目终端运行编译命令`cordova build ios`之后，再运行
